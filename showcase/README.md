# Showcase / tutorial world

A walkable **"Great Work" feature hall** that demonstrates every block and item in
*Octaves of the One*, grouped into wings that mirror the
[Great Work advancement tree](../docs/wiki/The-Great-Work.md).

- **Design spec:** [`docs/showcase-world-plan.md`](../docs/showcase-world-plan.md)
- **Source of truth:** the [`echoes_showcase` datapack](datapack/) — generated by
  [`gen_showcase.py`](../gen_showcase.py), which builds the hall via `mcfunction`s on world
  load. 61 stations (32 block displays + 29 item frames) across 8 wings, plus a welcome
  atrium and a few static working-build demos (ambient capture, Growth Radiator over crops,
  a wireless pair).
- **Ready-to-play save:** [`dist/Octaves-of-the-One-Showcase.zip`](dist/) — built and
  packaged by [`build_showcase.sh`](../build_showcase.sh).

## Play it

1. Install **Fabric Loader**, **Fabric API**, and **Octaves of the One** for **26.1.2**
   (the world is full of the mod's blocks, so the mod must be present).
2. Unzip `Octaves-of-the-One-Showcase.zip` into your `.minecraft/saves/` folder.
3. Open the world (**Creative**) — you spawn at the atrium; walk **east** through the wings.

## Rebuild it from source

```bash
python3 gen_showcase.py      # regenerate the datapack
./build_showcase.sh          # build on a headless server + repackage the save (JDK 25)
```

The datapack is plain data — drop it into any world's `datapacks/` folder and run
`/reload` then `/function echoes_showcase:build/all` to build the hall in-place
(`/function echoes_showcase:reset` clears it).

> The packaged save is built on a headless server, so it has no screenshots baked in —
> it's also the set used to capture the in-game gallery shots.
