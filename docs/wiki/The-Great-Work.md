# The Great Work

[← Home](Home.md)

**The Great Work** is the mod's in-game **advancement tree** — a guided path through the
whole progression, themed in Walter Russell's voice. Open your advancements (`L`) and find
the **Octaves of the One** tab; each step pops a toast and points at the next thing to
build, so you never need to memorize recipes or leave the game to read this wiki.

It mirrors the cosmology: matter is mined, wound up into Light, carried and banked, poured
back out as work, and finally transmuted — generation and radiation in balance.

## The tree

The root is mining your first **Raw Echocite**. From there the tree branches into the
mod's five strands. *Frames:* ◻️ task · ◆ goal · ★ challenge (challenges and goals grant
XP).

```
Octaves of the One (mine Raw Echocite)
├─ The First Tone (smelt → Echo Ingot)
│  ├─ A Winding Engine (Resonant Coil)
│  │  ├─ Banked Light (Resonance Cell) ─── Catch the Sky ◆ (Storm Caller)
│  │  ├─ Carried, Not Consumed (Wave Conduit) ─── Cut the Cord ◆ (Wave Relay)
│  │  ├─ Doubling Down (Compressor)
│  │  ├─ Fuelless Fire (Transmuter)
│  │  └─ Light as Life (Growth Radiator)
│  │     ├─ A Hearth of Light (Warmth Radiator)
│  │     ├─ Two Poles, One Field (Polarity Field)
│  │     └─ The Grid Breathes Evenly (Balancer)
│  └─ Over-Tuned (any Resonant tool)
│     └─ Where You Look ★ (Resonant Thrusters)
└─ The Unstruck Tone ◆ (find Silentite in the Deep Dark)
   ├─ The Still Centre (Stillness Core)
   └─ The Rest Point (Octave Seed)
      └─ Condensed Light ◆ (Radiant Ingot)
         ├─ High-Octave Bank (Greater Resonance Cell)
         ├─ The Higher Octave (Octave Coil)
         └─ Balanced Interchange ◆ (Transmutation Table)
            ├─ Carry the Light ★ (charge an Octave Star)
            └─ The Resolved Crest ★ (wind a Harmonic Mote)
```

## What each strand teaches

- **The grid (Echo Ingot → Coil → Conduit/Cell/machines).** The core energy loop:
  generate Light, carry it, bank it, and spend it on the Compressor and Transmuter. See
  [Energy System](Energy-System.md).
- **Radiation (Growth/Warmth Radiator, Polarity Field, Balancer).** The other half of the
  interchange — pouring Light back into the world as growth, heat, and motion.
- **Gear (Resonant tools → Thrusters).** Deliberately over-tuned tools and look-direction
  flight. See [Items & Gear](Items-and-Gear.md).
- **The octave climb (Silentite → Octave Seed → Radiant Ingot).** The high-octave tier:
  stronger generation, bigger banks, and the catalyst that opens transmutation. See
  [Crafting & Progression](Crafting-and-Progression.md).
- **The transmutation economy (Table → Octave Star / Harmonic Mote).** Dissolving and
  condensing matter as Bound Light. See [Transmutation & Light Values](Transmutation.md).

## For modpack makers

The tree is data only, under `data/echoes/advancement/great_work/`, and is regenerated by
[`gen_advancements.py`](https://github.com/Trystar360/echoes-of-the-deep/blob/main/gen_advancements.py)
(titles/descriptions live in the lang file under the `advancement.echoes.*` keys). Override
or extend it like any datapack — add nodes that parent onto `echoes:great_work/<node>` to
weave your pack's content into the same tree.
